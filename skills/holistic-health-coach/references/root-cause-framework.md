# 根因分析框架 / Root-Cause Framework

目的：从症状推到机制，而不是把建议堆在一起。讲透因果链，用户才能自己灵活搭配。

## 步骤

1. **症状聚类到系统**
   - 神经/认知：注意力↓、记忆↓、脑雾、拖延、刷短视频/游戏停不下来。
   - 代谢/胰岛素：嗜糖、餐后困、腹部脂肪、痘、能量起伏。
   - 皮肤/皮脂/炎症：出油、暗沉、爆痘、泛红、屏障受损。
   - 激素：月经/雄激素相关痘、甲状腺（怕冷/疲劳/体重）、压力-皮质醇。
   - 睡眠/昼夜节律：入睡难、夜醒、白天困、作息紊乱。
   - 营养缺乏：铁/维D/B12/omega-3/镁等。
   - 肠道：腹胀、排便异常、食物不耐受。

2. **为每个主诉列 2–4 个机制假设并排序**，写明判断依据（用户哪条信息支持/反对）。

3. **把习惯映射到机制**（常见因果链，均需按个体情况标注证据强度）：
   - 短视频/刺激游戏（可变奖励、高频强刺激）→ 中脑多巴胺系统对**慢速、低强度奖励脱敏** → 对学习/工作等"低多巴胺"任务难以启动、专注↓ [C，机制清晰但人类因果证据有限]。
   - 睡前蓝光 + 高唤醒内容 → 褪黑素延迟 + 交感兴奋 → 入睡难、深睡↓ → 次日记忆巩固↓、专注↓ [B]。
   - 慢性睡眠不足 → 前额叶功能↓、海马记忆巩固↓、胰岛素敏感性↓、皮质醇↑ → 认知↓ + 皮肤修复↓ + 嗜糖 [A]。
   - 高 GI / 高糖 + 乳制品（尤其脱脂奶/乳清）→ 胰岛素 + IGF-1↑ → 皮脂↑、毛囊角化↑、雄激素活性↑、炎症↑ → 痘 [B]。
   - 长期高 omega-6 / 低 omega-3（大量种子油+少鱼）→ 促炎类二十烷酸比例↑ → 低度炎症，可能加重痘/暗沉/情绪 [C，比例与总量、加工方式都重要，勿妖魔化"油"本身]。
   - 慢性低度炎症 + 糖化(AGEs) + 缺觉 + 紫外线 → 胶原降解 + 微循环差 → 皮肤暗沉、老化 [B]。
   - 缺铁/缺维D/缺B12 → 疲劳、脑雾、注意力↓、脱发、肤色差 [A，前提是确实缺乏]。

4. **指出可验证的检查/自测**：相关血检；2 周饮食+睡眠+屏幕日志；单变量戒断测试（如 2 周睡前不刷短视频看专注变化）。

5. **输出**：每个主诉给「最可能问题 → 怎么造成的（因果链）→ 底层逻辑（可迁移的原理）」。

## 推理纪律
- 区分**相关 vs 因果**；多数生活方式证据是观察性的，明说。
- 优先解释能同时影响多个主诉的**共同上游因素**（如缺觉、胰岛素抵抗、慢性炎症往往一因多果）——高杠杆点通常在这里。
- 不把单一食物/补剂神化或妖魔化；剂量、总膳食结构、个体差异才是关键。
- 任何"最可能"都标注不确定性，并给出能进一步确认的下一步。
