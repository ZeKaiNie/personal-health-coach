# 运动处方 / Exercise

> 运动同时改善认知、情绪、代谢、皮肤微循环、睡眠——一项干预多处获益。证据 [A/B/C/D]。

## 一、底层逻辑
- 有氧运动 → BDNF↑、海马神经可塑性↑、执行功能与情绪改善。[A/B]
- 力量训练 → 肌肉量↑→静息代谢↑、胰岛素敏感性↑、骨密度↑、抗衰老。[A]
- 运动改善内皮功能与微循环 → 利于肤色与组织修复。[B]
- 规律运动改善睡眠质量与入睡(但**剧烈运动别太靠近睡前**)。[B]

## 二、每周框架（WHO 指南为基线）[A]
- **中等强度有氧 150–300 min/周**，或高强度 75–150 min/周。
- **力量训练 ≥2 天/周**，覆盖主要肌群。
- 减少久坐：每坐 30–60 min 起身活动几分钟。[B]

## 三、模块
### 1. Zone 2 有氧（打底，提线粒体/代谢健康）
- 强度：能正常对话的微喘程度(最大心率~60–70%)。[B]
- 形式：快走、慢跑、骑行、椭圆机；每次 30–60 min，每周 2–4 次。

### 2. 力量训练
- 全身 2–4 次/周；复合动作(深蹲/硬拉/推/拉/髋铰链)优先。[A]
- 渐进超负荷：每个动作 2–4 组、6–15 次，接近力竭留 1–3 RIR。[A]
- 新手先学动作模式与姿势，再加重量，避免受伤。

### 3. 高强度间歇(HIIT)（时间效率高，按需）
- 如 4×4 min 高强度 + 间歇，或 10–20 min 间歇；每周 1–2 次，注意恢复。[B]

### 4. 日常活动(NEAT) + 步数
- 目标 ~7000–10000 步/日（个体化）；走路是最易坚持的代谢与情绪干预。[B]

## 四、和目标对齐
- **提升专注/情绪**：把一段中等有氧放在需要专注的工作前(急性增益)。[B]
- **减脂**：力量保肌 + 有氧 + 饮食热量缺口(见 nutrition.md)。[A]
- **增肌**：渐进超负荷力量 + 热量盈余 + 足量蛋白。[A]

## 五、恢复与安全
- 睡眠与蛋白是恢复的关键；练得动也要恢复得了。[A]
- 渐进增加(每周总量增幅别过大)，预防过用伤。[B]
- 有心血管疾病/久未运动/受伤史/孕期 → 开始前咨询医生。⚠️
