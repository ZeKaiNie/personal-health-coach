# 皮肤修复：暗沉 / 爆痘 / 屏障

> 适用：出油、爆痘、暗沉、泛红、屏障受损。证据标注 [A/B/C/D]。

## 一、底层逻辑

### 痤疮（痘）的四大致病环节
1. **皮脂分泌过多**（受雄激素、IGF-1、胰岛素驱动）。
2. **毛囊口角化异常**（堵塞形成粉刺）。
3. **痤疮丙酸杆菌（C. acnes）增殖**。
4. **炎症反应**。
→ 任何能降低胰岛素/IGF-1、雄激素活性、炎症、角化的措施都可能改善。[A 病理共识]

### 饮食与痘的关联（机制 + 证据）
- **高血糖负荷(高 GI/高糖) 饮食** → 胰岛素 + IGF-1↑ → 皮脂↑ + 雄激素活性↑ + 角化↑ → 痘。低 GL 饮食在 RCT 中改善痘。[B]
- **乳制品**（尤其脱脂奶、乳清蛋白）→ 含乳源 IGF-1 / 刺激胰岛素 → 与痘相关（观察性为主，脱脂奶关联强于全脂）。[B]
- **omega-6 过高 / omega-3 过低** → 促炎 → 可能加重炎性痘。[C]
- 注：巧克力、油炸的独立证据较弱；关键是**整体血糖负荷 + 乳制品 + 炎症**。[C]

### 暗沉 / 老化的机制
- **糖化(AGEs)**：高糖使胶原蛋白交联硬化、变黄 → 暗沉、失弹。[B]
- **慢性炎症 + 紫外线** → 胶原降解(MMP↑)、色素沉着、屏障受损。[A 紫外线]
- **缺觉 + 微循环差 + 脱水** → 气色差、黑眼圈。[B]
- **氧化应激**（吸烟、污染、缺抗氧化）→ 加速老化。[B]

## 二、护肤方案（简单有效 > 复杂）

### 核心三件套（适合大多数人）
1. **温和清洁**：氨基酸类洁面，早晚各一次；**避免过度清洁/皂基**（破坏屏障反而更出油爆痘）。[B]
2. **保湿修屏障**：含神经酰胺/甘油/透明质酸；屏障好，出油和敏感都会改善。[B]
3. **防晒**：每天 SPF30+，是**抗暗沉/抗老/防痘印加深**的最高杠杆护肤步骤。[A]

### 针对痘 / 暗沉的活性成分
| 成分 | 作用 | 用法/浓度 | 证据 | 注意 |
|------|------|-----------|------|------|
| 维A类(维A醇/视黄醛/处方维A酸) | 抗角化、抗痘、抗老、提亮 | 晚上，低浓度起步、建立耐受 | [A] | 孕期禁用；初期刺激/脱皮；必须配防晒 |
| BHA(水杨酸) | 溶解角栓、抗炎、控油 | 2% 局部/全脸，隔天起 | [A-B] | 敏感肌减频 |
| 壬二酸(Azelaic acid) | 抗痘+提亮+抗泛红 | 10–20% | [B] | 较温和，孕期相对安全(遵医) |
| 烟酰胺(Niacinamide) | 控油、修屏障、提亮 | 2–5% | [B] | 高浓度可能刺激 |
| 维C(L-AA) | 抗氧化、提亮、助胶原 | 早晨 10–20% | [B] | 不稳定，注意配方/避光 |
| BPO(过氧化苯甲酰) | 杀 C. acnes | 2.5–5% 局部 | [A] | 漂白衣物、刺激 |

> 原则：**一次只加一种活性，建立耐受**；刺激叠加是爆痘和屏障受损的常见原因。中重度/囊肿痘 → 看皮肤科（可能需口服药）。⚠️

## 三、饮食 / 生活方式（治本）
- 降**血糖负荷**：少含糖饮料/甜食/精制碳水，主食换全谷+足量蛋白纤维。[B]
- 试**减/换乳制品**（尤其脱脂奶、乳清）4–8 周看反应（个体差异大）。[B]
- 提高 **omega-3**（深海鱼/补剂），整体降炎症。[C]
- 睡眠 + 减压（皮质醇↑会加重痘和屏障问题）。[B]
- 戒烟、控酒；多彩蔬果（抗氧化、抗糖化）。[B]
- 不挤痘、不过度去角质。[B]

## 四、补剂（详见 supplements.md）
- **锌**（30–40 mg 元素锌/日，餐后，短期）对炎性痘有一定证据。[B] 注意长期高锌→缺铜。
- **omega-3**：抗炎。[C]
- 维生素D：缺乏者纠正可能改善痘。[C]
- 维C / 抗氧化：膳食优先。[C]

## 五、复盘
- 指标：每周新发痘数量、出油程度、暗沉(自评/拍照对比)、屏障敏感度。
- 单变量：先做"防晒 + 温和清洁 + 保湿 + 降糖"4 周；再引入一种活性成分；记录拍照对比。
