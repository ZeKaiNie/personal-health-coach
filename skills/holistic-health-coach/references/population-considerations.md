# 不同人群考量 / Population Considerations

> 同样的底层逻辑，按年龄/阶段/生理状态调整侧重与安全边界。先看记忆档案里的年龄与状态，再套用。

## 学生 / 青少年（中学–大学）
- 高频诉求：专注/记忆、熬夜、刷短视频、痘、外貌焦虑、考试压力。
- 侧重：睡眠规律(晚睡型常见，固定起床+晨光最关键)、屏幕宵禁、稳血糖早餐、运动提专注。
- 安全：**未成年人**慎用补剂(多从食物获取)；不给限制性减脂方案；外貌/进食焦虑要温和、避免制造焦虑，必要时引导心理支持。⚠️
- 生长发育期蛋白与总能量需求相对高，别盲目节食。

## 年轻成人（20–40）
- 高频诉求：健身增肌/减脂、皮肤、精力、代谢、压力、生育力。
- 侧重：力量训练+蛋白、稳定作息、酒精/咖啡因管理、皮肤抗老起步(防晒+维A)、代谢早筛(血糖/血脂)。
- 女性：经期相关症状、铁需求、孕期/备孕需单独按医嘱(很多补剂/活性成分孕期禁用，如口服/外用维A酸)。⚠️

## 中年（40–60）
- 侧重：代谢综合征/胰岛素抵抗筛查、血压血脂、肌肉量维持、压力与睡眠、激素变化(围绝经期)。
- 用药增多 → 补剂相互作用排查更重要。

## 老年人（60+）
- 高频诉求：养生、长寿、精力、认知、骨骼、慢病管理。
- 侧重：
  - **肌少症防治**：蛋白需求更高(常建议 1.2–1.6 g/kg，无肾病前提)、抗阻训练是核心。[A/B]
  - **维生素D + 钙 + 适度负重运动**护骨；**B12**吸收下降需关注。[A/B]
  - **认知与社交、规律作息、防跌倒**(平衡/力量训练)。[A/B]
  - 蛋白质分配到每餐(对抗合成抗性)。[B]
- 安全(高优先)：**多重用药(polypharmacy)** → 任何补剂都要查相互作用；肾/肝功能影响剂量；脱水/电解质风险；体重骤降/吞咽困难/跌倒/认知骤变是红旗→就医。⚠️
- 任何慢病(糖尿病/心血管/肾病等)管理以主治医生为准，本 skill 仅做生活方式辅助。

## 通用安全分流
- 孕期/哺乳/备孕、未成年人、确诊疾病、多重用药、红旗症状 → 见 `evidence-and-safety.md` 安全红线，优先建议专业评估。
