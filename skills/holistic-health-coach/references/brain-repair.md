# 大脑修复：注意力 / 记忆 / 多巴胺脱敏

> 适用场景：刷短视频/打刺激游戏后专注力↓、记忆↓、脑雾、对正经任务难启动。
> 证据标注 [A/B/C/D]，见 evidence-and-safety.md。

## 一、底层逻辑（先理解，再行动）

1. **多巴胺不是"快感分子"，是"动机/预测误差"分子**。短视频、刺激游戏靠**可变比率奖励 + 高频强刺激**反复拉高多巴胺峰值；大脑会**下调多巴胺受体/抬高基线阈值**来维持平衡。结果：高刺激内容才有"劲"，而看书、学习、工作这类**慢速、延迟、低强度奖励**变得"没意思、启动不了" → 主观专注力↓。[C：机制成熟，人体长期因果证据有限]
2. **注意力是可训练的资源**，被"碎片化、频繁切换"消耗。任务切换有**切换成本**，频繁查看手机会持续占用工作记忆。[B]
3. **记忆靠睡眠巩固**：海马把当天经历在**深睡(SWS)和 REM** 期转入皮层长期记忆；缺觉/碎片睡眠直接削弱记忆。[A]
4. **大脑可塑性**：神经元连接随行为重塑（"用进废退"），所以修复 = **减少脱敏刺激 + 重新训练慢奖励/深度专注 + 保护睡眠和血流**。[A 总原则]

## 二、修复策略（按杠杆从高到低）

### 1. 给多巴胺系统"降噪"（dopamine detox 的正确版本）
- 真正有用的不是"什么都不做"，而是**减少超常刺激、把奖励变难一点**。[C]
- 具体：
  - 短视频/刺激游戏设硬上限（如每天 ≤30–60 min，固定时段），用**摩擦**降低易得性（卸载、登出、灰度模式、App 限时）。[B 行为干预]
  - **晨起 1 小时内不刷**短视频/不碰高刺激内容——保护一天的动机基线。[D 经验]
  - 把"无聊"留给大脑：排队/走路不掏手机，让基线刺激回落。[C]

### 2. 重新训练深度专注
- **番茄钟 / 时间块**：从 25 min 起，逐步拉长到 50–90 min 单任务，中间不切屏。[B]
- 物理隔离手机（不同房间 / 抽屉）：仅"看不见"就显著降低认知负荷。[B]
- 单线程：关掉通知、一次只开一个任务窗口。[B]

### 3. 保护睡眠（记忆与专注的根基）
- 见 `sleep-circadian.md`。重点：规律作息、睡前 60–90 min 屏幕宵禁、保证深睡。[A]

### 4. 运动（最强的"促智"干预之一）
- 规律有氧 → BDNF↑、海马体积↑、执行功能↑。每周 150+ min 中等有氧 + 力量。[A]
- 单次中等有氧后**急性提升专注与情绪**，可放在需要专注的工作前。[B]
  详见 `exercise.md`。

### 5. 营养与血糖稳定
- 避免血糖大起大落（高 GI 早餐 → 餐后困、注意力↓）：足量蛋白 + 纤维 + 健康脂肪。[B]
- omega-3（DHA/EPA）支持神经膜与情绪；缺乏者补充获益更明显。[B]
- 纠正缺铁/缺维D/缺B12——这些缺乏直接导致脑雾/疲劳。[A 前提是缺乏]
  剂量见 `supplements.md`。

### 6. 压力与正念
- 慢性压力(皮质醇↑)损害海马与前额叶；正念/呼吸训练改善注意网络。[B]
- 自然光 + 户外 + 适度社交，改善情绪与昼夜节律。[B]

## 三、可放进方案的补剂（详见 supplements.md）
- **咖啡因 + L-茶氨酸**（100–200 mg + 200 mg）：提升专注同时削弱焦躁感。[B]
- **omega-3 (EPA+DHA)**：[B]
- **肌酸 5 g/日**：对睡眠不足/素食者的认知可能有益。[B-C]
- 纠正性补充：维D、铁（仅在确证缺乏时）、镁（睡眠）。[A/B]
- 谨慎对待"聪明药"/兴奋剂类——不在生活方式范畴，需医生。⚠️

## 四、4 周自测
- 指标：单次专注时长、每日短视频时长、主观脑雾(1–10)、入睡时间。
- 单变量实验：先只改"睡前 90 min 不刷短视频 + 手机离床"，两周看专注/睡眠变化，再叠加下一项。
